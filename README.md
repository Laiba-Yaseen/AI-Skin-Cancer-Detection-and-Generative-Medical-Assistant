# AI Skin Cancer Detection and Generative Medical Assistant

This project combines Deep Learning and Generative AI for skin cancer diagnosis.

Features:
- CNN-based skin cancer detection
- HAM10000 dataset
- Google Gemini AI integration
- Medical explanation generation
- Treatment recommendations
- Prevention guidance
- PDF medical report generation

Technologies:
- Python
- TensorFlow/Keras
- OpenCV
- Gemini API
- FPDF
- Google Colab
